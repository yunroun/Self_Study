<태그>
script : JavaScript를 사용할 수 있게 해준다.
input : 사용자의 데이터를 받을 수 있는 대화형 컨트롤을 생성한다.

<속성>
(type)
button : 버튼을 생성한다.
text : 사각형 칸이 생기고 그 안에 글씨를 입력할 수 있다.

value : 글을 작성할 수 있다.
alert : 경고창을 띄워준다.

<이벤트>
onclick : onclick 안 문법을 기억하고 있다가 클릭 시 이벤트를 실행한다.
onchange : 바뀌었을 때 이벤트를 실행한다.
onkeydown : 키를 눌렀을 때 이벤트를 실행한다.

<콘솔>
'~~'.length : 따옴표 안 문자의 개수를 알려준다.
'~~'.toUpperCase() : 알파벳 소문자를 대문자로 바꿔준다.
'~~'.indexOf('찾고자 하는 문자') : 찾고자 하는 문자가 몇 번째에 있는지 알려준다.
'   ~~   '.trim() : 문자에 있는 공백을 없애준다.
