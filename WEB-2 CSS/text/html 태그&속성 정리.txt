<태그>
font : 글씨의 색상을 변경해준다. (CSS로 인해 지금은 쓰이지 않음.)
div : 아무런 의미가 없고 디자인의 용도로 사용하는 태그. block level element.
span : div와 동일. inline element.
link : 다른 파일을 연결해준다.

<속성>
style : HTML 안에서 CSS를 사용할 수 있게 해준다.
class : 클래스를 지정해준다. (class로 지정해준 문단들을 하나로 묶어준다.) class로 지정해 준 선택자는 앞에 .을 붙여줘야 함.
id : class와 동일하지만, id > class. id로 지정해 준 선택자는 앞에 #을 붙여줘야 함.
